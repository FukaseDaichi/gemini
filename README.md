# Gemini を試す

## Next.js のプロジェクト作成

以下の`my-app`は好きな名前にする。

```
$ npx create-next-app@latest my-app --typescript --tailwind --eslint
```

## v0 の初期化

プロジェクトのディレクトリ（上記手順の`my-app`）で以下のコマンドを実行。

```
npx v0@latest init
```

## shadcn-ui の初期化

```
npx shadcn-ui init

```

以下追加の例

```
npx shadcn-ui@latest add button
```

## gemini の初期化

```
npm i @google-ai/generativelanguage
```
